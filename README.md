# PortSwigger Web Security Academy Writeups

This repository contains writeups for labs from the PortSwigger Web Security Academy.

The purpose of these writeups is to document the methodology used to identify and exploit common web vulnerabilities.

## Vulnerabilities Covered

- SQL Injection
- Cross-Site Scripting (XSS)
- Access Control Issues
- File Upload Vulnerabilities
